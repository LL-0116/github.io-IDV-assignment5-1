Portfolio Overview

What the page contains: This portfolio integrates Assignments 1 through 4 into a single cohesive interface, showcasing skills in API handling, SVG manipulation, interactive layout design, and D3.js data visualization.

How I merged the assignments: I utilized an `iframe` based architecture. Each assignment is kept in its original file to preserve its specific CSS frameworks (like Tailwind in A3) and JavaScript variables, ensuring zero conflicts between assignments.

Difficulties faced: The main challenge was ensuring that the height of the embedded documents adjusted correctly so that inner scrollbars didn't conflict with the main page scrollbar.

Hardest assignment to integrate: Assignment 3 was the most challenging because it is itself a container of multiple documents. Embedding a viewer inside a viewer required careful layout adjustment to ensure the inner charts remained visible.
